KEY FEATURES DOCUMENTED
=======================

1. Multi-Server Orchestration
   - Coordinate actions across multiple SSH servers
   - Web servers, app servers, databases, caches, etc.

2. Credential Isolation
   - AI agents never see IPs, passwords, or SSH keys
   - Only friendly host names exposed

3. Self-Discovery
   - Agents discover available servers automatically
   - Servers advertise their capabilities

4. Dual Deployment Modes
   - STDIO Mode: Local deployment (Claude Desktop, VS Code)
   - HTTP Mode: Remote deployment (ChatGPT, web clients)

5. Authentication Options
   - API Key authent, Okta, Keycloak)
   - JWT token validation with JWKS

6. Complete Auditability
   - All commands logged with timestamps
   - User identity tracking (from OAuth tokens)
   - Full command history

7. Production Ready
   - Docker deployment
   - Health checks
   - Session management with auto-cleanup
   - Resource limits
   - Non-root execution
